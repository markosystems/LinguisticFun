# LinguisticFun
The following library is partially experiemental, but also part of a game that I'm working on.
One part of the library makes the words more phonetic, the other part applies different standards and rules that either don't make since or come from an entirely different language. 

There's a Library, a server, and a chrome extention.
The extention talks to the server to grab the different spellings of words then replaces all text with the new spelling.

Of course, the phonetic side is mostly complete other than some edge-cases because I'd have to hardcode for those.
